<p>The Infix Calculator implements a three-phase processing pipeline that transforms user-entered mathematical expressions into computed results. The system accepts infix expressions containing single-digit numbers, arithmetic operators, and parentheses, then produces floating-point results with four decimal places of precision.</p>
<p><strong>Key Capabilities:</strong></p>
<ul>
<li>Infix to postfix expression conversion using the Shunting Yard algorithm</li>
<li>Postfix expression evaluation using stack-based arithmetic</li>
<li>Input validation for bracket matching and operator sequences</li>
<li>Support for operator precedence rules (^, *, /, +, -)</li>
<li>Interactive command-line interface with error recovery</li>
</ul>
<p><strong>Technical Constraints:</strong></p>
<ul>
<li>Single-digit operands only (0-9)</li>
<li>Fixed-size arrays for stack storage</li>
<li>No support for decimal input numbers</li>
<li>Recursive error handling approach</li>
</ul>
